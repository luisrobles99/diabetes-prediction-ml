# 🏥 Predicción de Diabetes con Machine Learning

## 📋 Descripción
Modelo de clasificación para predecir diabetes en pacientes
usando datos médicos reales del dataset Pima Indians.

## 🎯 Resultados
| Modelo | Precisión |
|--------|-----------|
| Regresión Logística | 75.32% |
| SVM | 74.68% |
| Random Forest | 73.38% |
| KNN | 72.08% |

## 🔍 Hallazgo principal
La glucosa es el predictor más importante (27% de importancia),
seguido del BMI (17%) y la edad (14%).

## 🛠️ Tecnologías
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## 📊 Dataset
Pima Indians Diabetes Database — Kaggle
768 pacientes, 8 variables médicas
